# Oil Amor — Enterprise Grade

> *Essential oils that transcend consumption.*

A cinematic, luxury-grade website for Oil Amor featuring biophotonic aesthetics, magnetic interactions, and horizontal scroll galleries.

---

## ✨ Experience Highlights

### Visual Design
- **Biophotonic Color System** — Miron Violetglass deep purples with aurum gold accents
- **Grain Texture Overlay** — Animated film grain for tactile, analog luxury feel
- **Glassmorphism Effects** — Translucent layers with backdrop blur
- **Editorial Typography** — Cormorant Garamond display + Inter body

### Interactions
- **Magnetic Cursor** — Custom cursor that morphs based on context (view, add, external)
- **Magnetic Buttons** — Elements subtly follow cursor on hover
- **Smooth Scroll** — Locomotive Scroll integration with velocity-based skew
- **Horizontal Gallery** — Draggable, scrollable product atelier
- **Parallax Effects** — Multi-layer depth on hero section

### Animations
- **Cinematic Preloader** — Letter-by-letter reveal with progress bar
- **Scroll-Triggered Reveals** — Elements animate into view as you scroll
- **Counter Animations** — Stats count up with easing when visible
- **Continuous Micro-animations** — Floating crystals, pulsing orbs, orbiting rings

## 🎨 Design System

### Colors
| Token | Value | Usage |
|-------|-------|-------|
| `--miron-void` | `#0a0612` | Deepest background |
| `--miron-base` | `#1a0f2e` | Primary dark |
| `--miron-glow` | `#5a3d8c` | Accent glow |
| `--gold-pure` | `#c9a227` | Primary accent |
| `--gold-light` | `#e8d5a3` | Highlight text |
| `--light-cream` | `#f8f6f3` | Light backgrounds |

### Typography Scale
- **Hero**: 6rem / Cormorant Garamond / Weight 300
- **Section Title**: 3.5rem / Cormorant Garamond / Weight 300
- **Body**: 1rem / Inter / Weight 400
- **Eyebrow**: 0.6875rem / Inter / Weight 600 / Uppercase / Letter-spacing 0.25em
